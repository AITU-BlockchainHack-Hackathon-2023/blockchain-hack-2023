## Blockchain Hack 2023 Front-end

## Installation
To install and set up the libraries, run:
```sh    
$ npm install
```

## Usage

### .env example (./)
```env
APP_URL= localhost:9000
API_URL= http://localhost:8080/api/v1
```

### Serving the app in development environment

```sh
$ npm start
```

### Building a distribution version

```sh
$ npm run build
```
